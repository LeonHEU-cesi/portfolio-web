## Why a blog

When looking for a development job, showing code isn't always enough. A blog is a way to share how you reason, why you pick one tech over another, and what you learn along the way.

## What to expect

- Short **technical notes** on bits of code that surprised me.
- **Experience reports** from professional or personal projects.
- **Mini-tutorials** when I struggled to find a clear answer.

## How this site is built

Astro 5 with 100% static output, Tailwind CSS 4 in CSS-first config, Markdown content via Astro Content Collections. Source code [on GitHub](https://github.com/LeonHEU-cesi/portfolio-web).
